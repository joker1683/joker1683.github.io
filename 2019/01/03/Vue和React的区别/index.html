Vue和React的区别

React数据不可变,单向数据流
vue是基于可变数据的，支持双向绑定
React是类式的组件写法
vue是声明式的写法
React做的很少，很多由社区来完成
Vue内置了很多功能，致力于开发简单
React是通过JS操作一切
Vue是基于单组件模版
