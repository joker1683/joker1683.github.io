算法 

找出数组 arr 中重复出现过的元素

function duplicates(arr) {
var result = [];
arr.forEach(function(elem) {
if(arr.indexOf(elem) !== arr.lastIndexOf(elem) && result.indexOf(elem) === -1) {
result.push(elem)
}
})
return result;
}

深拷贝和浅拷贝

深拷贝:function deepCopy(source,copy) {
var copy = copy || {};
for(var i in source) {
if(typeof source[i] === “object”) {
copy = source[i].constructor === “Array”?[]:{}
} else {
copy[i] = source[i]
}
}
return copy;
}
浅拷贝:function shallowCopy(source) {
var copy = {};
for(var i in source) {
copy[i] = source[i]
}
return copy;
}

Http协议

简述http协议：
http协议是无状态的，
http包括持久连接和非持久连接：
非持久连接：每次传输都需要单独建立一次TCP连接
持久连接：只需建立一次TCP连接，客户端和服务器可一直请求和响应
TCP/IP 三次握手:客户端发出SYN包加一个序号m，服务端收到之后将收到的序号加一并发送ACK(m+1)＋SYN包带一个新产生的序号n一同返回给客户端，客户端收到之后将序号加一并返回ACK(n+1)包
四次挥手:主动方发出FIN包带一个序号m，关闭主动方的连接，被动方收到之后发送ACK包带序号m+1，被动方再次发送FIN包并带一个序号,关闭被动方的连接，主动方收到后发送ACK包并将收到的序号加一
网络七层模型:物理层 数据链路层 网络层 传输层 会话层 表示层 应用层
TCP四层模型:数据链路层 网络层 传输层 应用层
缓存:强制缓存 协商缓存 查看缓存相关头信息cache-control expires看是否满足强制缓存，如果不满足携带缓存相关头信息请求服务器，服务器查看之后是告知直接从缓存读取还是返回最新的资源内容。
http2和http1.1的区别:
1.1 的头信息是文本格式的，2的头信息和数据都是二进制的。
2可以复用TCP连接，如果一个请求的数据量过大，可以先把已处理的数据先返回给客户端，然后暂时挂起这个请求，先接收别的请求,最后再回头处理这个耗时的请求
2可以给每个数据包一个标识，数据包可以不按顺序发送，取消发送数据的时候可以发送一个信号，而不用关掉TCP连接
2可以让服务器主动推送给客户端资源，例如一些静态资源
2头信息用gzip进行压缩，并且客户端和服务端共同维护一张头信息表，所有字段都会生成一个索引号，以后就不用 发送同样的字段了
